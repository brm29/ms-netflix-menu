# Netflix's Greatest Shows

The goal of this project was to create a **menu-style webpage** that displays a collection of items organized into categories. I chose to create a showcase of some of the greatest Netflix series, grouped by genre and theme.

## Project Overview

This website shows a collection of popular shows on Netflix and Netflix original shows, organized into multiple categories. Each show is displayed as a card containing its title, rating, genre, description, and poster image. The design is inspired by Netflix's dark interface while maintaining a clean and responsive layout.

## Technologies Used

* HTML5
* CSS3

## Features

* Multiple categories of TV shows
* Responsive card-based layout
* Show ratings and genre information
* Netflix-inspired dark theme
* Hover effects for cards and images
* Mobile-friendly design
* Clean and well-organized code structure

## What I Learned

This project gave me the opportunity to reinforce good front-end development practices.

Throughout this project, I focused on:

* Organizing content into clear, user-friendly categories.
* Building a consistent and responsive card-based layout.
* Writing cleaner, more maintainable CSS by grouping related styles and using descriptive comments. As stylesheets grow, well-placed comments make the code easier to navigate and maintain.
* Using Git for version control by making regular commits throughout development, making it easier to track progress and manage changes.
* Improving project organization through a clear folder structure, meaningful file naming, and concise project documentation with a README.
